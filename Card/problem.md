#

給定 $N$ 張牌，每張牌從 $1$ 至 $N$ 依序編號。
給定牌的初始排列，輸出經過 $K$ 次的洗牌後的排列。

每次洗牌，位置 $i$ 的牌位置變為 $F(i)$，保證：

 - $1 \leq F(i) \leq N$
 - 對於任意的 $i \neq j$，$F(i) \neq F(j)$

$F$ 函數會直接給定 $N$ 個映射關係

\clearpage

## 輸入
第一行有兩個整數 $N, K$，表示牌的數量和洗牌次數。
第二行有 $N$ 個整數，為牌的初始排列。

## 輸出
輸出一行有 $N$ 個整數，以空格隔開，為最後的牌的順序。

## 輸入限制
 - $1 \leq N \leq 300000$
 - $0 \leq K \leq 10^{18}$

## 子任務
| 編號 | 分數 |    限制    |
| --- | --- | ---------- |
|  1  |  | 初始序列就是$1$至$N$，$N \leq 10$，$K = 1$ |
|  2  |  | $N \leq 10$，$K \leq 100$ |
|  3  |  | $N \leq 10$ |
|  4  |  | $N \leq 100$ |
|  5  |  | $N \leq 3000$ |
|  6  |  | 無額外限制 |

\clearpage
